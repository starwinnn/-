🎨 像素战痘记 (Pixel Acne Log)
一个超可爱的纯前端面部皮肤记录工具 ✨

不想用复杂的 APP 记日记？试试这个像素风的小工具吧！它把枯燥的“抗痘”过程变成了一个有趣的像素游戏。
复制链接，浏览器打开，投射到手机桌面即可使用
🌟 核心功能
👩 像素捏脸交互：告别枯燥的文字，直接在可爱的像素女孩脸上点击，哪里长痘点哪里！

🚦 痘痘进化论：独特的“生命周期”记录法。

👇 点 1 下：🔴 红肿（刚冒头）

👇 点 2 下：⚪ 白头（成熟啦）

👇 点 3 下：🟣 痘印（快好啦）

👇 点 4 下：🗑️ 消除（拜拜！）

📅 战痘年历：内置 2026 年热力图日历，每天长了几颗一目了然，直观看到皮肤变好的过程。

🔒 绝对隐私：没有后台，不需要注册。所有数据都保存在你自己的手机浏览器里（LocalStorage），只有你能看到。
